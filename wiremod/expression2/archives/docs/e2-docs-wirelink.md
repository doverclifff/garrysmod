### hispeedReturnValue(![Number](Type-Number.png "Number") Value)

If used from inside of event readCell, satisfies the incoming read request with the provided number.
Clears HiSpeed error flag. (2 ops)

### hispeedSetError(![Number](Type-Number.png "Number") Value)

If used from inside of event readCell or event writeCell, non-zero indicates that the read/write request couldn't be fulfilled.
Doesn't error your chip. (2 ops)
